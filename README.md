This is a learning project intended to teach me the solidity programming language and how to deploy smart contracts
This is not a serious project and might contain some errors

1. Users can enter the lottery with a predetermined amount of ETH based on the current ETH/USD price
2. An admin will choose when the lottery is finished
3. The lottery will select a random winner
